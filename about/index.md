## How to Deploy?

### 1. Clone the project

```bash
git clone https://github.com/Glomzzz/Rhapsody.git
```

### 2. Rhapsody, Start!

```bash
cd Rhapsody
npm install
npm run dev
```

Of course, you can seamlessly switch to pnpm, yarn, or even bun.
